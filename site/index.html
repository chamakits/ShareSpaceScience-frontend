Triangle Cloud Foundry Meetup

On Tuesday, October 21, the Triangle Cloud Foundry Meetup group will be holding a Hands on with Cloud Foundry meeting at IBM in RTP, NC. Directions in the previous link.

The IBM developers at the meeting will probably be using Bluemix as their Cloud Foundry platform of choice. You might want to register for a free account (using previous link) before the meeting.

As an additional set up item, you may want to install the cf tool, from here: https://github.com/cloudfoundry/cli#downloads
